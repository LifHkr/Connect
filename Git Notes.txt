## Help Command

	$ git help
		f forward
		b backwards
		q quit


##Where is Git Config


System:

		Linux		/etc/gitconfig
		Windows		Program Files\Git\etc\gitconfig
		
		
User:

		Linux		~/.gitconfig
		Windows		$HOME\.gitconfig
	
	$ cat .gitconfig	
		

Project

		my_project/.git/config
